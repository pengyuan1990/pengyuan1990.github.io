从头开始css

人老了，记性不好了，那就记笔记吧！

css中长度和百分比的含义

em的意义

  • em：是相对于当前元素的字号进行大小匹配的。实际上可以把“em”当作字号的大小。

%的意义

  • %：是相对于其父元素中相同属性的大小来进行匹配的。

line-height中纯数组的意义

  • line-height中纯数字，表示是根据当前元素的字号进行匹配的。参考

伪类&伪元素

  • 伪类：对应一组html元素，无须在html中用类标记

    • 伪元素：在html中不存在的元素

    字体样式

    • 斜体

    • 粗体

    • 字体大小

    行高设置

    • 属性值为数字：如果使用数字设置行高，那么所有的子元素会自己继承这个数字

    • 属性值为em或者%：子元素只会继承计算后的值

    背景设置

    背景具有多个属性值：background-color,background-image,background-repeat,background-attachment,background-position,background-size等。

    • background-color：用于设置背景颜色

    • background-image：设置背景图片

    • background-repeat：背景图片的重复方向

    • background-attachment：背景图片是否随页面滚动

    • background-position：指定背景图片的位置

    间距控制

    通过控制单词间距和字母间接(字偶距)调整文本距离。

    • word-space： em/px. em会根据当前元素字体大小计算出结果。子元素之后继承计算之后的结果
    • letter-space： em/px.

      缩进

    • text-indent: em/px;

      文本大小写

    • text-transform:uppercase,lowercase,none,capitalize(每个单词首字母大写)

      小型大写字母

    • font-variant:small-caps,none

      文本装饰

    • text-decoration: underline(下划线),overline(上划线),line-through(删除线),none

      空白属性

    • white-space:用于控制是否忽略额外的空格

    如果overflow:hidden;text-overflow:ellipses。则会显示省略号...

    文本换行

    • word-wrap: break-wrod;
