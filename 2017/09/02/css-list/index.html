web列表样式细节

有序列表&无序列表

ol和ul分别代表有序和无序列表标签。列表在显示的时候默认具有一定的缩进可以通过margin-left:0,进行取消，在实际应用中无序列表多一些。

选择标记

无论哪种列表，都可以选择出现在列表左侧的标记类型。可以通过list-style-type进行选择。

  • disc 圆点
  • circle 圆圈
  • square 方块
  • decimal 数字1，2，3
  • upper-alpha 大小字母
  • lower-alpha 小写字母
  • upper-roman 大写罗马数字
  • lower-roman 小心罗马数字

    自定义标记

    • 在设置自定义标记之前，需要将list-sytle设置为none
    • 为ul(ol)添加margin-left:0或者padding-left:0，清理默认缩进，然后根据需求为li元素添加padding-left属性。

tips：通过list-style-image也可以添加自定义标记，但是由于浏览器兼容性的问题，尽量自己实现。

标记位置

默认情况下，标记位于整个文本块的外面，看起来标记和文本是两个分离的部分。可以通过list-type-position:inside/outside进行设置。

form相关

fieldset和legend

fieldset是用来对form中的元素信息进行分组的，以明确form中不同部分的功能。在每个fieldset中可以使用legend属性为该分组添加标题。默认情况下fieldset具有边框。
Tips：其中legend必须包含在fieldset中。

label

标签是用于描述表单字段的文本。标签的for元素和表单字段中的id关联起来，如果两个的值相同两者就具有关联关系.

select

通过optgroup可以对select的信息进行分组.通过label属性为分组添加文本标题
