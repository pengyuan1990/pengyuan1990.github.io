你不知道的js,读书笔记.

作用域

JavaScript是属于动态或解释执行语言.但js也具有编译过程,只是js的编译和执行过程相隔十分近.对于js来说大部分情况下编译发生在代码执行的前几微秒。
Tips:抽象语法树(AST):将词法数组转换为一个由元素逐级嵌套所组成的代表了语法结构的树。

  • 引擎：从头到尾负责整个js的编译执行过程。
  • 编译器：负责语法分析以及代码生成。

  • 作用域：负责收集维护所有声明的标识符组成的一系列查询，并根据相关查询规则确定当前执行的代码对标识符的访问权限。

  • LHS&RHS：表示赋值操作的左查询和右查询

    作用域嵌套

    当一个块或者函数嵌套在另一个块或者函数中是，就发生了作用域嵌套，当在当前作用域无法找到变量是引擎就会在外层继续查找，直到找到或者抵达最外层嵌套。

    词法作用域

    词法作用域：词法定义阶段的作用域，词法作用域是由写代码时变量和作用域块的位置决定的。

  • 查找：在找到第一个匹配的标识符停止，在多久嵌套中内层标志符可以屏蔽外层同名标识符。
    TIPS：无论函数在哪里被调用或者如何调用，它的词法作用域都是只由函数声明是所处位置决定的。！

  • 词法作用域查找只会查找一级标识符。

  • eval&with
    eval：函数可以接受一个字符串作为参数，并将其中的内容视为存在于程序中的代码一样处理。

    函数作用域&块作用域

  • function为一句话的第一个单词为函数什么，否则为函数表达式
    function a(){} // 声明
(function a(){}()) //表达式
  • js中能够定义块作用域的有：with，try-catch中的catch模块，let，const
  • click事件添加会在函数内部形成一个闭包，注意内存泄漏的问题
